# Gorilla-tag-REPO
Gorilla tag repositories - this contains every version since  24 Jan. 26

NOTE:
This is **NOT** a crack of Gorilla Tag. This is a fresh release REPO (unopened). These are old but still playable versions of gorilla tag that will not work unless...
A) You own the game on Steam
B) It's the current version

If it is not the current version, you won't be able to join rooms. If you don't own the game on steam, Steam will literally restrict you from opening the executable entirely.
Please use these responsibly. If this repo is used irrationally, the blood is on your hands, not mine. Do what you will, but all the stupid shit **YOU** do is completely on you.

Enjoy!


If you want to do this on your own...


Go to any web browser and type:

Steam://open/console

then go to:

https://www.SteamDB.info

Make sure Steam.DB is signed in (with steam)

Search "Gorilla Tag"

Click Gorilla Tag

go to 'Packages' on the left side

Click 'Gorilla Tag' under 'Store Packages'

go to 'Depots' on the left side

Click the only option

Go to 'Manifests' on the left side

find the update you want

copy the package using the copy button on the far right

go to the console you opened before in steam

type download_depot 

paste the copied package 

remove -app, -depot, -manifest but keep the numbers

it should download to:

?:\your\drive\Steam\SteamApps\Content\app_1533390

open the correct depot

run Gorilla Tag.exe as you usually would


Voila!
